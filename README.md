# Gustari-autentica

## Description

Gusturi Autentice is a web application developed using React and TypeScript, designed to showcase and manage traditional food products. The application provides an intuitive interface for browsing products, filtering information, and interacting with a structured catalog of traditional Romanian foods.

## Features

* Product listing and management
* Product filtering and search functionality
* Responsive user interface
* Data validation
* Modular React component architecture
* TypeScript support for improved code quality

## Technologies Used

* React
* TypeScript
* Vite
* CSS
* Node.js

## Installation

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project folder:

```bash
cd food-store
```

3. Install dependencies:

```bash
npm install
```

4. Run the application:

```bash
npm run dev
```

## Project Structure

* `src/components` – reusable UI components
* `src/pages` – application pages
* `src/hooks` – custom React hooks
* `src/context` – state management
* `src/utils` – utility functions
* `tests` – unit tests
